# ⚡ SiteCloner — High-Fidelity Website Cloning Engine

SiteCloner is a premium, AI-powered tool designed for developers and designers who need **high-fidelity clones** of modern websites. While achieving 100% pixel-perfection on framework-heavy sites is an industry-wide challenge, SiteCloner uses a next-gen headless engine and AI vision to achieve **70-80% visual and functional fidelity out of the box.**

## 🚀 Key Features

*   **Fidelity Healing Engine**: Uses **Gemini 2.5 Vision** to bridge the gap from 70% to 100% by identifying and patching visual regressions, broken paths, and structural errors.
*   **Intelligent UI Cleanup**: Automatically removes blocking, broken elements like Disclaimer banners and non-functional "Agree" popups using the **V6 Ultra** "Janitor" logic.
*   **Auto-Scroll Discovery**: Dramatically improves image capture by simulating user interaction to trigger lazy-loaded assets before serialization.
*   **Deep CSS Extraction**: Aggressively captures many keyframes and font-faces, though complex animation-heavy sites may require a second AI pass.
*   **Quick Preview**: Generates a local `run.bat` for every clone for one-click launching.

## 🛠️ Tech Stack

*   **Engine**: Puppeteer (Headless Chrome)
*   **Backend**: Node.js / Express
*   **Intelligence**: Google Gemini 2.5 & DeepSeek V3
*   **Frontend**: Professional Glassmorphic Dashboard

## 💎 Current State & Growth

SiteCloner is an evolving engine. While it handles standard layouts with near-perfect accuracy, modern complex frameworks like Next.js often present hydration challenges. Our **AI Finisher** is specifically designed to recognize these failures and apply surgical code patches, aiming to push your clone's fidelity towards the 100% mark.

## 🔧 Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/mustadafinshimanto/SiteCloner-v1.5.git
    cd SiteCloner
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Launch the app:
    ```bash
    run-sitecloner.bat
    ```
